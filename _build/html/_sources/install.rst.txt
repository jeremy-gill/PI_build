Installation
============

From source::

  $ python setup.py install

Using pip::

  $ pip install -e git+git://git@git.parallax-innovations.com/jgill72/PI_build/#egg=PI_build

As an entry in a :file:`requirements.txt` file of a different project::

  git+ssh://git@git.parallax-innovations.com/jgill72/PI_build/#egg=PI_build
