PI_build Docs¶

This python package contains tools to automatically maintain version info for python software packages that are kept in git repositories. Tag projects with names that look like vA.B.C, and PI_build can be used to automatically populate version strings elsewhere. This package depends primarily on dulwhich, which is used to directly query software repositories.
